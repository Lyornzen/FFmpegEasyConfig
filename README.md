# FFmpegEasyConfig

现代化桌面端 FFmpeg 图形界面，基于 Electron + React + TypeScript + Ant Design。

## 功能

- **视频** — 格式转换、剪辑、合并、分割、压缩、提取帧、旋转、水印
- **音频** — 格式转换、剪辑、合并、提取、音量调节、背景音乐、降噪
- **图片** — 图片转视频、视频转 GIF、GIF 优化
- **流媒体** — RTMP 推流、HLS 转换、录屏
- **高级** — 命令生成器、批处理、GPU 编码、自定义参数

## 开发

```bash
npm install
npm run dev      # 启动开发模式
npm run build    # 编译
npm run package  # 打包为安装包
```

## 技术栈

Electron 28 · React 18 · TypeScript 5 · Ant Design 5 · Vite 5
