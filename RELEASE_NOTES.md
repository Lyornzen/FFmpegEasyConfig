# FFmpeg UI v1.0.0

🎉 首个正式发布版本！一款基于 Electron + React + TypeScript 的现代化 FFmpeg 桌面 GUI 工具。

## ✨ 功能一览

### 🎥 视频处理
| 功能 | 说明 |
|------|------|
| 格式转换 | MP4 / AVI / MKV / MOV / WebM 等常见格式互转 |
| 视频压缩 | CRF / 码率 / 分辨率 / 编码器多维度压缩控制 |
| 视频剪辑 | 按起止时间截取片段 |
| 视频拼接 | 多视频合并 |
| 视频旋转 | 90°/180°/270° 旋转及镜像翻转 |
| 提取帧 | 按时间或帧率提取为图片 |
| 转 GIF | 视频片段转高质量 GIF |
| 添加水印 | 图片水印 + 文字水印 |

### 🎵 音频处理
| 功能 | 说明 |
|------|------|
| 格式转换 | MP3 / WAV / FLAC / AAC / OGG 互转 |
| 音量调整 | 放大 / 缩小 / 归一化 |
| 音频提取 | 从视频中提取音频 |
| 降噪处理 | 背景噪音消除 |
| 音频拼接 | 多音频合并 |
| 背景音乐 | 为视频叠加背景音乐 |

### 🛠️ 高级功能
- **批量处理** — 多文件批量执行同一操作
- **GPU 加速** — NVENC / AMF / QSV 硬件编码
- **HLS 转换** — 视频切片为 m3u8
- **自定义参数** — 自由编辑 FFmpeg 命令行
- **命令生成器** — 可视化生成 FFmpeg 命令
- **GIF 优化** — 体积与质量平衡优化
- **图片转视频** — 图片序列合成视频

### 📡 推流
- **RTMP 推流** — 实时推流到 RTMP 服务器
- **屏幕录制** — 屏幕内容实时推流

### 📁 字幕
- **添加字幕** — 嵌入 SRT / ASS / VTT 字幕
- **字幕转换** — 字幕格式互转
- **字幕提取** — 从视频提取字幕流

## 🖥️ 界面特性
- 无边框窗口设计
- 左侧导航栏快速切换
- 实时 FFmpeg / FFprobe 状态检测
- 任务队列管理（进度追踪、取消）
- 深色 / 浅色主题切换
- 日志面板

## 🔧 系统要求
- **操作系统**: Windows 10 / 11（x64）
- **Node.js**: ≥ 18（仅开发时需要）
- **FFmpeg**: 需安装并添加到 PATH，或通过应用内手动选择路径

## 📦 下载
- **安装包**: `FFmpeg-UI-Setup-1.0.0.exe`
- **便携版**: `FFmpeg-UI-Portable-1.0.0.exe`（免安装，即开即用）

## 🚀 快速上手指南
1. 下载安装包或便携版
2. 启动应用 → 首页自动检测 FFmpeg 环境
3. 选择功能页面 → 选择媒体文件 → 配置参数 → 执行
4. 实时查看进度和日志

## 🐛 已知问题
- macOS 和 Linux 构建将在后续版本支持
- 部分 GPU 编码器需要对应显卡驱动
